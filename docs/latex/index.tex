\hypertarget{index_autotoc_md6}{}\doxysubsection{Branching Strategy}\label{index_autotoc_md6}
The branching strategy will consist in 4 branch types\+:
\begin{DoxyItemize}
\item {\ttfamily main} The stable version of the project
\item {\ttfamily develop} Branched from main. Merged into main when becomes a stable product. ~\newline

\item {\ttfamily feature} Branched from develop for each feature in development. Merged into develop when finished.
\item {\ttfamily hotfix} Branched from main to fix bugs. Merged into main when the bug is fixed. 
\end{DoxyItemize}
